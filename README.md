# Download4Me

Парсер для скачивания файлов из интернета по настраиваемым правилам.

Настройка парсинга осуществляется через SQLite: задает сайты-источники, страницы для обработки и правила извлечения ссылок на файлы.
После скачивания страниц правила применяются к содержимому — найденные ссылки добавляются в очередь на скачивание.
